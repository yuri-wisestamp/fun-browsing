Welcome to fun and safe browsing!
====

Running
---

Local server:\
`cd server`\
`npm run serve`

Extension:\
in Chrome, open  <a href="chrome://extensions">chrome://extensions</a>\
Turn *developer mode* on\
Click *Load unpacked*\
navigate to the *extension* folder and pick it
